### 快速开始
###### 运行环境
- Java-21
- Nodejs-20.12.2
###### **项目安装**

1. 拉取项目至本地 [https://example.com](https://example.com)
```sh
git clone https://git.summer.fan:9443/jack/hmoneta.git
```
2. 项目依赖安装
  - 后端依赖安装
```sh
mvn dependency:copy-dependency
```
  - 前端依赖安装
```sh
cd /project/web
yarn
```
